# B-Roll Video Prompts for ADP GCP Observability Dashboard

This document contains 50 detailed video prompts optimized for Google Veo 3.1 text-to-video generation. These videos are designed for use on the landing/login page of the ADP GCP Observability Dashboard application.

**Output Directory:** `client/public/videos/`  
**Recommended Duration:** 10-15 seconds per video (seamless loop preferred)  
**Recommended Resolution:** 1920x1080 (16:9) or 4K  
**Color Palette:** Deep blues, ADP red (#d0271d) accents, white highlights, dark backgrounds

## Usage Notes

### Generation Settings for Google Veo 3.1
- **Aspect Ratio:** 16:9 (1920x1080 or 3840x2160)
- **Duration:** As specified per prompt (10-15 seconds)
- **Style:** Professional, enterprise, technology-focused
- **Quality:** High fidelity, cinematic rendering
- **Loop:** Where noted, ensure seamless loop capability

### Implementation
1. Generate videos using prompts above
2. Export in MP4 format with H.264 or H.265 codec
3. Save to `client/public/videos/` directory
4. Reference in login page component for background rotation

### Color Reference
- **Primary Blue:** #1e40af (Deep enterprise blue)
- **ADP Red:** #d0271d (Brand accent)
- **Success Green:** #22c55e
- **Warning Amber:** #f59e0b
- **Error Red:** #ef4444
- **Background:** #0a0a0f (Near black)
